# Kids Learn to Code

A Python quiz & flashcard app to help kids (and beginners) learn to code.

Built by **Grace Kabaghe** as a hands-on way to learn Python — one session at a time.

## What it does (so far)
- Greets the user and asks their name.
- (More coming: quiz questions, scoring, flashcards, categories!)

## How to run it
```
python3 quiz.py
```
